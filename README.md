## Run Locally

Clone the project

```bash
  git clone https://github.com/dark-venom26/Adyoda-Assignment
```

Go to the project directory

```bash
  cd Adyoda-Assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Tech Stack

**Client:** React, SASS

**Server:** Node
